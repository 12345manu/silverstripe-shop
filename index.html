SilverStripe Shop ecommerce shopping cart solution

The SilverStripe Shop module aims to provide developers with a framework for building, and customising ecommerce-based projects.

The SilverStripe Shop module provides facilities for:

  • browsing and managing a product catalog
  • placing and managing orders
The module is not designed to be a boxed package (at least not at this stage). A strong focus is being put on enabling developers to create custom online shopping solutions. The core has been kept to a minimum, and additional features are provides as 'submodules'.

Dependencies

  • SilverStripe version 2.4+
  • Payment Module version 0.3+

Install

  • Obtain the module code, and put into a 'shop' directory, within your SilverStripe root directory.
  • Do the same for the 'payment' module.
  • Run yoursite/dev/build?flush=all
